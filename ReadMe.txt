 Instructions followed for installing the code and using 
* First install Arduino IDE with latest version
*Then, download the "final_code.ino"
*By configuring the pins as per the description in the code we can able to establish the communication between the arduino and the FlySky i6 Reciever.
* "Quadruped.ino" file is a backup code where we tried and demonstrated on various walking algorithms for smooth motions.
* Ardupilot latest firmware is to be downloaded from the Ardupilot website.
